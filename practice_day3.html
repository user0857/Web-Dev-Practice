Element Arrangment Practice

By default, element in HTML is arranged vertically with top-down fashion. However, if we would like the element being arranged otherwise, float: none/left/right/initial/inherit should be added to the style container. Notice that float controls whether an element should float after setting a value to it. To initialize the float, use clear: both to push element below left and right floated elements.

Margin Design

If we would like to arrange element with certain margin between each other, we can set it up in such fashion: {margin: up_pixel, right_pixel, down_pixel left_pixel}, there are other ways to achieve the same funcitonality, but in short, this is the most understandable and cleverest way to set up margins from 4 direction in my opinion.

Illustration

Let's say we would like 4 bricks be saperated by 30 pixels to each other, and these bricks are aligned horizontally from left to right, we can add margin: 0(px), 15px, 0(px), 15px to the style container to do the job.

Padding Design

In the previous section, we've shown that margin help seperating elements in certain specification. However, we haven't learned how to generate spaces INSIDE an element. To achieve this, we can use {padding: up_pixel, right_pixel, down_pixel left_pixel;}

Block and Inline Elements

A block element always starts are a new line, and browser will add margin on left and right side. The width (100% by default) and height can be explicitly set up. <p> and <div> are great example of block elements.
On the other hand, inline element does not start on a new line and only takes up minimum width necessary. Take <span> for instance.
