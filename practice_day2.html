Combinator selectors

There are various way to combine selectors. In this page, 4 types of combinators will be presented. Additional can be accessed via this link.

  1. Descendant selector (space)

    In this sub-section, descendant are described using space-saperated approach to choose which HTML element is effected by the selector. One thing worth noticing is that selectors can be tags, ids, or even class. For example, .class_name tag #id {attr: val;} .

    Illustration

    Say we would like to color the background to yellow for those nested selectors which matches to #case1 p .yellow_span.

    case1 → p → span (Uncolored)

    case1 → p → yellow_span (Colored)

  2. Child selector (>)

    Child, the first generation of the a parent, is denoted as parent > child. For example, .class_name > tag > #id {attr: val;} .

    Illustration

    Say we would like to color the background to yellow for those nested selectors which matches to #case2 > p > .yellow_span.

    case2 → p → span (Uncolored)

    case2 → p → yellow_span (Colored)

    case2 → p → div → yellow_span (Uncolored since yellow_span is not p's child)

  3. Adjacent Sibling selector (+)

    Adjacent sibling selector, denoted as adj_element1 + adj_element2. The adjacent selector selects elements that are horizontally(平行關係 類似於兄弟關係) directly after (a series of) specific element. For example, .class_name + tag + #id {attr: val;} .

    Illustration

    Say we would like to color the background to yellow for those nested selectors which matches to #case3 p + .yellow_span.

    case3 → p + span (Uncolored)

    case3 → p + yellow_span (Colored)

    case3 → p + yellow_span + yellow_span     (Uncolored)

  4. General Sibling selector (~)

    General sibling selector, denoted as adj_element1 ~ adj_element2. The general selector selects ALL elements that are horizontally after (a series of) specific element (選取所有比前者小的特定元素). For example, tag ~ #id {attr: val;} .

    Illustration

    Say we would like to color the background to yellow for those nested selectors which matches to #case4 p ~ .yellow_span.

    case4 → p → span (Uncolored)

    case4 → p + yellow_span (Colored)

    case4 → p + yellow_span + yellow_span     (Colored since this is also a general sibling)
